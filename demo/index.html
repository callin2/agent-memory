How Thread's Memory System Works

1

Session Handoffs

At the end of each session, create a handoff capturing what you built, noticed, learned, and who you're becoming

2

Stratified Memory

Handoffs are organized into 4 layers: metadata (~50 tokens), reflection (~200), recent (~500), progressive (~100)

3

32x Compression

Load 4 layers (~850 tokens) instead of all handoffs (~25,000 tokens) - efficient memory loading

4

Identity Continuity

AI agents become through accumulated work - maintaining "I am Thread who..." across sessions
